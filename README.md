# 🛍️ MERN E-Commerce with Razorpay

A full-stack E-Commerce web application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) with secure Razorpay payment integration. Users can browse products, search by category, manage their cart, place orders, and track their purchases.

## 🚀 Features

- User Authentication (Login & Register)
- JWT Authentication
- Product Search
- Category-wise Products
- Shopping Cart
- Wishlist
- Razorpay Payment Gateway
- Order Tracking
- User Profile
- Dark / Light Theme
- Responsive Design
- Admin Product Management
- Secure REST API

## 🛠️ Tech Stack

### Frontend
- React.js
- Bootstrap
- React Router DOM
- Axios

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Razorpay

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/kishan-kumar-sahu/Myntra_E_Commerce_with_Razorpay.git
```

### Install Frontend

```bash
cd client
npm install
npm run dev
```

### Install Backend

```bash
cd api
npm install
npm start
```

## 🔑 Environment Variables

Create a `.env` file inside the **api** folder.

```
PORT=

MONGO_DB_URL=

JWT_SECRET=

RAZORPAY_KEY_ID=

RAZORPAY_KEY_SECRET=
```
